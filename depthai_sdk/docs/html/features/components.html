Components¶

Components are part of the OakCamera class and abstract DepthAI API nodes; their initialization, configuration, and linking. This improves ease of use when developing OAK aplications.

Available components

  • CameraComponent

  • NNComponent

  • StereoComponent

  • IMUComponent

Reference

class depthai_sdk.components.Component¶

SDK component is used as an abstraction to the current DepthAI API node or group of nodes.

forced_openvino_version()¶

Checks whether the component forces a specific OpenVINO version. Only used by NNComponent (which overrides this method). This function is called after Camera has been configured and right before we connect to the OAK camera. @return: Forced OpenVINO version (optional).

Return type

Optional[depthai.OpenVINO.Version, None]

abstract on_init(pipeline, device, version)¶

This function will be called after the app connects to the Device
