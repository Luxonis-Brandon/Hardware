Code Samples¶

Code samples are used for automated testing. They are also a great starting point for the DepthAI API, as different node functionalities are presented with code.

ColorCamera- Demonstrates how to control the RGB camera (crop, focus, exposure, sensitivity) from the host

  • Auto Exposure on ROI - Demonstrates how to use auto exposure based on the selected ROI

  • RGB Camera Control

Mixed

  • RGB Encoding & MobilenetSSD - Runs MobileNetSSD on RGB frames and encodes FUll-HD RGB into .h265 and saves it on the host

  • RGB Encoding & Mono & MobilenetSSD - Runs MobileNetSSD on mono frames and displays detections on the frame + encodes RGB to .h265
